# job_salary_prediction_dataset.csv
Job Salary Prediction Dataset

📊 Project Overview

This project focuses on analyzing job salary trends using Excel. The dataset includes various factors such as experience level, company size, and job roles.

🎯 Objective

- Analyze salary trends based on experience
- Compare salaries across company sizes
- Visualize insights using pivot tables and dashboards

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization Dashboard

📁 Dataset

The dataset contains:

- Job Title
- Salary
- Experience Level
- Company Size
- Location

📈 Key Insights

- Higher experience leads to higher salary
- Large companies tend to pay more
- Salary varies by job role and location

🚀 How to Use

1. Download the CSV file
2. Open in Excel
3. Use Pivot Tables for analysis
4. Explore dashboard insights

📌 Future Improvements

- Add Power BI dashboard
- Build machine learning model for prediction
